# 🏫 小小地方官 - AI 社會實踐互動教材

這是一款專為國小中低年級設計的互動學習工具，透過「小小地方官」的角色扮演，引導學生學習社會科課綱中的「公共參與」與「資源調度」概念。


## 🎮 立即體驗
[點擊這裡開始遊戲](https://weddy1225-blip.github.io/AI-social-study05/)

---



## 🌟 教學目標

本教案設計結合 AI 評分機制，幫助學生建立以下核心素養：
- **優先順序決策**：理解生命安全（Safety）相對於其他社會事務的急迫性。
- **多元公共事務認知**：認識鄰里紛爭、硬體建設、與文化資產保存等不同面向。
- **公民責任與溝通**：透過 AI 角色（里民、秘書長）的反饋，體驗公共決策後可能面臨的社會期待與壓力。

## 🎮 遊戲特色

- **200 題完整題庫**：包含安全、鄰里、建設、文化四大類別各 50 題，題庫豐富且不重複抽題。
- **AI 模擬反饋**：系統會隨機出現小學生、陳奶奶、張先生等角色，根據學生的決策給予人情味十足的回應。
- **直覺式操作**：採用拖拉式介面，完美適配平板電腦（iPad）與行動裝置教學環境。
- **在地文化整合**：題庫融入台灣本土文化背景，如原民祭典、老厝保存與社區生態議題。

## 🛠️ 技術架構

- **前端開發**：HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla JS)。
- **核心邏輯**：
  - 非同步狀態追蹤，確保 50 回合內題庫不重複。
  - 響應式佈局 (Responsive Design)，支援教室平板與大螢幕教學。
- **佈署平台**：GitHub Pages。

## 🚀 如何開始使用

1. 點擊本專案的 **[GitHub Pages 連結](https://[您的GitHub帳號].github.io/little-official/)** 即可開始。
2. 將畫面上的陳情卡片拖曳至「第 1 優先」到「第 4 順位」的格子中。
3. 點擊「提交決策」，閱讀 AI 秘書長與里民的建議。
4. 點擊「處理下一批陳情」練習更多案例。
